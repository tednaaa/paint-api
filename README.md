# Project init

> Set env vars and install packages

```bash
cp .env.example .env
npm ci
```

> Compiles and hot-reloads for development

```bash
npm run dev
```

> Compiles for test production build

```bash
npm run build
```

> Lint project and find mistakes

```bash
npm run lint
```

> Compiles for deploy on heroku

```bash
npm run heroku-postbuild
```

> Start nodejs server

```bash
npm start
```
