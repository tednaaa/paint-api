# Project init

> Set env vars and install packages

```
cp .env.example .env
npm ci
```

> Compiles and hot-reloads for development

```
npm run dev
```

> Compiles for test production build

```
npm run build
```

> Lint project and find mistakes

```
npm run lint
```

> Compiles for deploy on heroku

```
npm run heroku-postbuild
```

> Start nodejs server

```
npm start
```
